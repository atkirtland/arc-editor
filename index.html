Quick Tips

  • Number Keys (0-9): Press any number key to quickly select a color from the palette
  • Left-Click & Drag: Click and drag your mouse to paint multiple cells at once
  • Right-Click: Flood fill a connected region with the selected color
  • Ctrl+Left-Click: Replace all instances of the clicked color in the current grid with the selected color
  • Ctrl+Right-Click: Replace all instances of the clicked color in all grids with the selected color
  • Toggle: Click a cell with the same color to reset it to black (0)
  • Auto-Save: Your work is automatically saved to your browser and will persist between page refreshes
