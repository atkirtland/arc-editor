# arc-editor

completely vibe-coded arc puzzle editor

Hosted at <https://kirt.land/arc-editor>.

Judge Cursor (auto model) for code quality, not me

`json_to_images.py` is for producing PNGs from the JSON files. It will produce 4 of them (training only, training+test input, test input only, test output). The website also has a PNG exporter for the two most useful of these.

Let me know if you'd like any changes.
